# BMI Calculator

## Description
A beginner-friendly Python project that calculates Body Mass Index (BMI) based on user input for weight and height, then classifies the result into standard health categories.

## Features
- Takes weight and height as input.
- Accepts height in feet and inches.
- Converts height to meters for BMI calculation.
- Calculates BMI using the standard formula.
- Classifies BMI into health categories.
- Handles invalid input gracefully.

## Formula Used
BMI = weight (kg) / height (m)²

## How to Run
1. Make sure Python is installed.
2. Save the file as `bmi_calculator.py`.
3. Open VS Code or terminal.
4. Run the program using:

```bash
python bmi_calculator.py
```

## Example
- Weight: 70 kg
- Height: 5 feet 8 inches
- Output: BMI value and health category
